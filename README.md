# 📸 Tradutor de Imagens com OCR

Aplicação Full-Stack que usa Visão Computacional para extrair texto de imagens (como cardápios) e traduzir para português.

## 🚀 Tecnologias
- **Back-end:** Python & Flask.
- **OCR:** Pytesseract (Tesseract Engine).
- **Front-end:** HTML5, CSS3 e JavaScript (Fetch API).

## ⚙️ Execução
1. `brew install tesseract`
2. `pip3 install flask pytesseract pillow`
3. `python3 app.py`
